# Student Performance Prediction

# Project Overview

This project uses Machine Learning to predict student math scores based on reading and writing scores.

# Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-Learn

# Machine Learning Workflow

1. Load dataset
2. Perform Exploratory Data Analysis (EDA)
3. Select features and target
4. Split data into training and testing sets
5. Train a Linear Regression model
6. Evaluate model performance using Mean Absolute Error (MAE)
7. Predict math scores for new students

# Results

Mean Absolute Error (MAE): 7.35

Example Prediction:

Reading Score: 80

Writing Score: 75

Predicted Math Score: 74.29
